# Americat

Learning some Python and Svelte

![Screenshot animated](screenshot.png "Screenshot")

### Start presentation
Download via [slides.pdf](slides.pdf) or start via 
```
yarn slides:start
npm run slides:start
```

### Start App

```
yarn start
npm run start
```

### Install App

```
yarn install
npm install
```
